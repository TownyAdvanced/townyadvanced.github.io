Wars: we gottem

Event War

Event War is built into Towny.jar. It is begun using "/ta toggle war" and presently is restricted to nation vs nation wars and requires an admin or commandblock to activate.

Event War does not persist between reloads/server stops. Event war is very versatile as far as configuration and allows for completely non-destructive wars for money, up to wars involving complete territorial control and money.

Event War will be recieving a significant update expanding its possibilities towards player-initiated war, war perisistance across server restarts, and wars involving non-nation-vs-nation entities.

Flag War

Flag War is built into Towny.jar. When the config "allow_attacks" is true all enemy-nations will be able to start nation vs nation combat with territory control enabled.

Flag War is 24/7 and can be very punishing in regards to towns being able to suffer tremendous loss of territorial control.

Flag War is transitioning towards becoming a separate plugin run along-side Towny.jar.

Siege War

Siege War is available as a separate plugin, allowing for player-enabled, multi-day long Sieges.

Sieges are conducted by Nations on Towns resulting in a geo-political battle for influential territorial control. Sieges are non-destructive of town lands and only come with monetary punishments, negated with the Towny Bankruptcy system.

Siege War was created by Goosius1 over the course of just over one year and is maintained by LlmDl.
