Towny Advanced Portal

Note: This portal page serves as a method of finding the various projects which assemble the Towny Advanced plugin ecosystem.

Each plugin exists as a separate repository on Github, with downloads available on the repository Releases page, and (usually) with its own readme/wiki/documentation.

Towny Advanced Organization's Plugins:

Plugins officially maintained and supported by Towny Advanced.

Towny

Our core plugin. Provides player-controlled region management, a resident/town/nation heirarchy, a vast API and an incredibly thorough feature-set resulting from 13 years of development.

Github Downloads Support Wiki Javadoc Spigot Page Wars Video Tutorials
TownyChat

Optional plugin to provide chat channels for towns, nations, local, global, staff; as well as chat formatting that includes PlaceholderAPI support.

Github Downloads Support
TownyFlight

Optional plugin for Towny-enabled flight.

Github Downloads Support Spigot Page
TownyProvinces

Optional plugin, which makes claiming more organized, reducing staff workload and server toxicity.

Github Downloads
TownyCultures

Optional plugin to give towns the ability to choose a culture.

Github Downloads
TownyResources

Optional plugin to give Towns and their Nations (and occupiers,) a daily resource, similar to Sid Meier's Civilization.

Github Downloads
Dynmap-Towny

Optional plugin to display Towns on Dynmap.

Github Downloads
MapTowny

Optional plugin to display Towns on SquareMap, Pl3xMap, Dynmap & BlueMap.

Github Downloads
TownyCombat

Optional plugin that adds improvements to combat-focused servers. Recommended to keep a level playing-field.

Github Downloads
WorldGuard-Towny

Optional plugin which adds Town-Creation flag to WorldGuard regions.

TownyNameUpdater

Required for servers in online mode, prior to Towny 0.96.5.2, for when players change their MC name.
